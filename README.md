## Overview
A comprehensive web-based harmonium trainer designed to help students of Indian Classical Music develop perfect pitch recognition and swar identification skills through interactive games and practice modes.

## Features

### üéπ Practice Mode
- **Complete 3-octave harmonium** with all 36 notes
- Visual distinction between Shuddha (white keys) and Komal/Teevra swars (black keys)
- Covers Mandra, Madhya, and Taar Saptaks
- Authentic harmonium sound synthesis with rich harmonics

### üéÆ Four Game Modes
1. **Classic Mode** - Single note identification for beginners
2. **Speed Challenge** - Time-based training (10 seconds per note)
3. **Sequence Memory** - Listen and repeat 3-5 note patterns
4. **Interval Training** - Identify the distance between two notes

### ‚öôÔ∏è Customization

- **3 Difficulty Levels**:

  - Medium: All 12 swars (includes Komal & Teevra)

### üèÜ Gamification

## Audio Engine
Custom harmonium sound synthesis featuring:
- Multiple oscillator layers for rich timbre
- Natural harmonic overtones
- Slight detuning for authentic chorus effect
- ADSR envelope for realistic attack and release
